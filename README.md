# Student Ranking System - DSA Project

Full-stack web application implementing sorting algorithms and binary search.

## Quick Start

1. Install dependencies:
```bash
cd server && npm install
cd ../client && npm install
```

2. Setup MongoDB Atlas connection in `server/.env`

3. Run servers:
```bash
# Backend: cd server && npm run dev
# Frontend: cd client && npm run dev
```

4. Access: http://localhost:3000

## Features
- 4 Sorting Algorithms (Merge, Quick, Bubble, Insertion)
- Binary Search
- Real-time statistics
- MongoDB Atlas backend
- React + Vite frontend

See full documentation in README files.
